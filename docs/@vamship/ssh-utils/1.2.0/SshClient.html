run(commands) → {Promise.<SshExecutionResults>}

Source:
  • ssh-client.js, line 157
Connects to the remote host, and executes the specified command(s) on it sequentially. Execution will be aborted if any of the commands fail.
Parameters:
Name Type Description
commands String | Array The commands to execute on the remote host.
Returns:
A promise that will be resolved with an object that defines the results of execution of the commands. Note that the promise will be resolved even if command execution fails. Promise rejection implies a general failure, such as connection errors, etc.
